The plugin simply adds spiders and bugs that run across your site - the perfect halloween-decoration for your site!

== Description ==
With this plugin you can decorate your site for Halloween with just one click - an Easter Egg that will delight your visitors!
There is an options page under settings where you can activate and throttle spiders and flies. These will appear on the page and crawl across the screen in a very real way. A spooky affair perfect for Halloween. 

== Installation ==
Two possibilities: 
1. via the Wordpress plugin \"Store\"
2. download ZIP file and unpack it in the folder wp-content > plugins

== Frequently Asked Questions ==
Can I only activate spiders? Yeah, sure!
Can I adjust the number of animals? Yes you can. Just go to the options page and choose between 6 different levels
